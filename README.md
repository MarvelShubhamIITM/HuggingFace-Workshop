# HuggingFace-Workshop
# Machine Learning Model Deployment with Gradio

A simple web application that demonstrates machine learning model deployment using a custom-trained Random Forest classifier.

## Project Overview

This project includes:
1. **Model Training**: Trains a Random Forest classifier on synthetic data
2. **Web Interface**: Provides an interactive web UI using Gradio
3. **Model Deployment**: Shows how to deploy ML models with HuggingFace Spaces
